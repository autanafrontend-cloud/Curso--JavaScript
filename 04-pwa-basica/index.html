Demo 4

App offline con Service Worker

Ejemplo moderno para llevar el concepto de AppCache a proyectos reales.

Conexión activa

Qué queda disponible sin red

  • La interfaz principal
  • Los estilos y el JavaScript
  • Esta tarjeta de contenidos
  • La página offline personalizada

Patrón explicado

  1. El navegador registra el service worker
  2. Durante install se cachean archivos
  3. En fetch sirve desde cache cuando falla la red
  4. La experiencia sigue viva incluso offline

Notas

La nota se guarda con localStorage, mientras que los archivos estáticos van por Cache API.
